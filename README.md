## Deskripsi Aplikasi
Aplikasi Daftar Pesawat adalah mini project Flutter yang berfungsi sebagai katalog sederhana untuk menyimpan informasi pesawat. Pengguna dapat menambahkan, melihat, mengedit, dan menghapus data pesawat dengan mudah. Aplikasi ini dibuat sebagai latihan dasar implementasi CRUD (Create, Read, Update, Delete) menggunakan Flutter.

## Fitur Aplikasi
- Create → Tambah data pesawat baru
- Read → Tampilkan daftar pesawat dalam bentuk list
- Update → Edit data pesawat yang sudah ada
- Delete → Hapus data pesawat dari daftar
- Multi Page Navigation → Navigasi ke halaman form input/edit

## Widget yang Digunakan
- Scaffold → Struktur dasar halaman
- AppBar → Header aplikasi
- ListView.builder → Menampilkan daftar pesawat
- Card & ListTile → Menampilkan item pesawat dengan rapi
- FloatingActionButton → Tombol tambah data
- TextField → Input data pesawat (Nama, Maskapai, Tahun Produksi)
- AlertDialog / Navigator → Form input/edit dengan navigasi antar halaman
- IconButton → Tombol edit dan delete
